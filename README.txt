September 28th 2015

Client for applications connecting to IBM BlueMix.

1) Create the API key in BlueMix IoTF using the IBM BlueMix Web Portal.
2) Copy authentication information in app.properties
3) Run BlueMixAppClientTest as an example

In case of proxy issues, make sure to pass your proxy configuration to the Java VM:
ex: -Dhttp.proxyHost=proxy.crc.ca -Dhttp.proxyPort=3128 -Dhttps.proxyHost=proxy.crc.ca -Dhttps.proxyPort=3128
